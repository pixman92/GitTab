Starting off:

Type "git" (in the omni bar), then press Tab. Type any of the commands:
commit -m <message>,
checkout <branchName>,
checkout -b <branchName>,
current

Committing

To commit, or add a brief message about, a series of tabs (or a branch),
"commit -m <your meessage here>"
 The message doesn't have to have ".

Create New Branch

To create a new 'branch' (series of saved tabs), simply type command: "checkout -b <branchName>".

Retrieve an Old Branch

To retrieve an alreay created 'branch', type: "checkout <branchName>".

Status

To access your history of commits and current tabs, type "Status", into the omnibox.

Current

current - alerts of the currentbranch
